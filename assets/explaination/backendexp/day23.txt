    🔧 Common Errors & Fixes
    Error	Cause	Fix
    403 on POST /api/pets	Missing or expired token	Login again and use fresh token
    Empty list on GET /api/pets	No pets with AVAILABLE status	Verify pets were added
    500 on GET /api/pets/999	Pet doesn't exist	Expected
    500 on DELETE with adopter token	Ownership validation	Expected
    status is null	Missing pet.setStatus("AVAILABLE")	Add before save
    shelter is null	Missing pet.setShelter(shelter)	Add before save
